### Запуск проекта
****
 Проект использует инструмент - just(https://just.systems) — современный и удобный runner команд.
 * Для macOS
```bash 
  brew install just
 ```
* Запуск
```bash 
  just run
 ```
* UI
```bash
  just ui
```
